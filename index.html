GeoFire

Our geospatial library provides functions to store data for location querying in Firebase, and performs location queries such as location updates and localized searches.

Firepad

An open source real-time collaborative text editor. It provides true collaborative editing, complete with intelligent operational transform-based merging and conflict resolution.

GitHub | Demo

Firefeed

Firefeed is a web app that lets users post small messages called sparks to their feed. You can follow other users, and their sparks will appear on your feed.

GitHub | Demo

Firechat

Firechat is an open-source, real-time chat widget built on Firebase. It offers fully secure multi-user, multi-room chat with flexible authentication, moderator features, user presence and search, private messaging, chat invitations, and more.
